# Text Annotations (Тестовое задание)

Одностраничное приложение (SPA) на Angular для создания статей и выделения фрагментов текста с добавлением пользовательских аннотаций.

Проект выполнен в рамках тестового задания с полным соблюдением ограничений на использование сторонних UI-библиотек.

## Основной функционал

- **Управление статьями (CRUD):** Создание, редактирование, удаление и просмотр списка статей.
- **Аннотирование текста:** Выделение любого фрагмента текста мышью, добавление текстовой заметки и выбор цвета подчеркивания.
- **Всплывающие подсказки (Tooltips):** При наведении курсора на выделенный текст отображается всплывающая подсказка с сохраненной заметкой.
- **Локальное сохранение (Mock Server):** Все данные (статьи и аннотации) сохраняются в `localStorage` браузера, имитируя работу с реальным бэкендом.

## Технологический стек

- **Фреймворк:** Angular 19 (использование Standalone Components, Control Flow синтаксиса `@if`/`@for`).
- **Язык:** TypeScript.
- **Управление состоянием:** RxJS (`BehaviorSubject`, `Observable`).
- **Формы:** Reactive Forms.
- **Стилизация:** Чистый CSS с использованием нативных CSS-переменных (без препроцессоров и библиотек вроде Bootstrap/Material).

## Как запустить проект локально

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (версия 18+).
2. Клонируйте репозиторий:
   git clone https://github.com/serj7220/text-annotations.git
3. Установите зависимости, которые описаны в файле pakage.json:
   npm install
4. В терминале выполните команду:
   npm run start
5. Перейдите в броузере по ссылке http://localhost:4200/
